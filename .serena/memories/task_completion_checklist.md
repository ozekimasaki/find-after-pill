# タスク完了チェックリスト

## コード変更後の必須チェック
- [ ] ESLintエラーがないことを確認 (`npm run lint`)
- [ ] TypeScript型エラーがないことを確認
- [ ] ビルドが成功することを確認 (`npm run build`)
- [ ] 開発サーバーで正常に表示されることを確認

## 機能追加時のチェック
- [ ] 新機能が仕様通りに動作する
- [ ] 既存機能に影響がないことを確認
- [ ] レスポンシブデザインが維持されている
- [ ] エラーハンドリングが適切に実装されている

## データ関連タスク
- [ ] データ取得スクリプトが正常に動作する
- [ ] KVストアへのデータ更新が完了している
- [ ] APIエンドポイントが正常に応答する

## デプロイ前チェック
- [ ] 全てのリントエラーが解消されている
- [ ] ビルドが成功している
- [ ] 環境変数が正しく設定されている
- [ ] KV Namespaceが正しく設定されている

## コミット前チェック
- [ ] コードがプロジェクトのコーディング規約に従っている
- [ ] 不要なconsole.logが削除されている
- [ ] コミットメッセージが適切である
- [ ] 変更内容がテストされている
